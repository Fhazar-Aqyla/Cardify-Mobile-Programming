Repo ini ditujukan agar mudah di-clone, dijalankan, dan dikembangkan oleh orang lain tanpa ribet.

### 1. Clone repository

```bash
git clone https://github.com/Fhazar-Aqyla/Cardify-Mobile-Programming.git
```

Masuk ke folder project:

```bash
cd Cardify-Mobile-Programming
```

---

### 2. Install dependency

Gunakan npm (default & aman):

```bash
npm install
```

Atau jika terbiasa pakai yarn:

```bash
yarn install
```

---

### 3. Jalankan aplikasi

```bash
npx expo start
```

Opsi menjalankan:

* Scan QR Code dengan aplikasi **Expo Go** (Android / iOS)
* Tekan `a` untuk Android Emulator
* Tekan `w` untuk Web
